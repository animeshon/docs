VoiceOver is any voice that is not part of a character (e.g. inner voice, narrator, etc.)

```graphql
type VoiceOver implements Metadata, Voiced {
  voices(
    filter: VoiceActingFilter
    order: VoiceActingOrder
    first: Int = 0
    offset: Int = 0
  ): [VoiceActing!]!
  id: String!
  selfLink: String!
  createdAt: DateTime!
  modifiedAt: DateTime!
  version: String!
  names(
    filter: TextFilter
    order: TextOrder
    first: Int = 0
    offset: Int = 0
  ): [Text!]!
  aliases(
    filter: TextFilter
    order: TextOrder
    first: Int = 0
    offset: Int = 0
  ): [Text!]!
  description(
    filter: TextFilter
    order: TextOrder
    first: Int = 0
    offset: Int = 0
  ): [Text!]!
  voicesAggregate(filter: VoiceActingFilter): VoiceActingAggregateResult
  namesAggregate(filter: TextFilter): TextAggregateResult
  aliasesAggregate(filter: TextFilter): TextAggregateResult
  descriptionAggregate(filter: TextFilter): TextAggregateResult
}
```
