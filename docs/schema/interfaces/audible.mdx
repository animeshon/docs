Audible contents have both soundtracks and voice actings
Episode, Anime, Drama CDs, Concerts....

```graphql
interface Audible {
  soundtracks(
    filter: SoundtrackFilter
    order: SoundtrackOrder
    first: Int = 0
    offset: Int = 0
  ): [Soundtrack!]!
  voiceActings(
    filter: VoiceActingFilter
    order: VoiceActingOrder
    first: Int = 0
    offset: Int = 0
  ): [VoiceActing!]!
  soundtracksAggregate(filter: SoundtrackFilter): SoundtrackAggregateResult
  voiceActingsAggregate(filter: VoiceActingFilter): VoiceActingAggregateResult
}
```
