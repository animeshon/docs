# Add Playlist Item

This page shows how to create a playlist item using Animeshon's GraphQL gateway.
To create the playlist check [Create a Playlist](mutation-create-playlist)'s page.

## Sample

Replace `[PLAYTLIST]` with an existing playlist.

Replace `[RESOURCE-NAME]` with a valid content in Animeshon's Knowledge Base.

:::tip

Try use `animes/4287922344858850139` as `[RESOURCE-NAME]`

:::

:::warning

The caller must be already authenticated.

:::

:::tip

Visit [https://animeshon.com/graphql](https://animeshon.com/graphql) to browse the GraphQL embedded documentation.

There is a single unified endpoint for Animeshon GraphQL queries and mutations: [https://animeshon.com/graphql](https://animeshon.com/graphql).

:::
