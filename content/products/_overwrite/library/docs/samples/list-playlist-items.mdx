# List items of a playlist

This page describes how to list the items of a playlist.

## Sample

The following sample shows how to list the items of a playlist.

Replace `[PLAYLIST]` with the resource name of the playlist the items will be listed from, e.g. `users/123/playlists/456`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
