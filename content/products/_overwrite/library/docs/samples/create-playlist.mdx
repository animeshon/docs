# Create a playlist

This page describes how to create a playlist.

## Sample

The following sample shows how to create a playlist.

Replace `[USER-OR-ORGANIZATION]` with the resource name of the user or organization that the playlist belongs to, e.g. `users/123` or `organizations/123`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
