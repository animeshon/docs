# List cross-references

This page describes how to list cross-references.

## Sample

The following sample shows how to list cross-references.

:::tip

It is possible to filter out only cross-references of a particular collection using `Prefix` filter.

```
Prefix: "animes" -> only animes/- cross-references
```

:::

:::tip

It is possible to filter out only cross-references with a pending edge (therefore waiting for an operator to take action) using the `PendingOnly` filter.

:::

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
