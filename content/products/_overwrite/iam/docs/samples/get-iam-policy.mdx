# Get IAM policies

This page describes how to programmatically retrieve policies assigned to an Identity and Access Management API resource.

## Sample

The following code sample shows how to retrieve policies assigned to an Identity and Access Management API resource.

Replace `[SERVICE-ACCOUNT]` with the resource name of the service account that the policy should be retrieved from, e.g. `users/123/serviceAccounts/my-service-account`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::

:::caution

The `SetIamPolicy` method will always replace (overwrite) any previously defined policy.

:::
