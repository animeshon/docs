# Create service account

This page describes how to create a service account.

## Sample

The following sample shows how to create a service account.

Replace `[NAME]` with the desired service account name, e.g. `users/123/serviceAccounts/my-service-account`.

Replace `[DISPLAY-NAME]` with the desired service account display name, e.g. `My Service Account`.

Replace `[DESCRIPTION]` with the desired service account description, e.g. `A test service account.`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
