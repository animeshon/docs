# Get credentials

This page describes how to get existing credentials, the retured value will not include username/password or tokens.

## Sample

The following sample shows how to get existing credentials, the retured value will not include username/password or tokens. Actual credentials cannot be retrieved by users or service accounts, their access is restricted for security reasons to internal services only.

Replace `[CREDENTIALS]` with the desired credentials, e.g. `users/123/credentials/myanimelist-net`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
