# Exchange with third-party services

This page describes how to perform an exchange with third-party services through OAuth 2.0 flows.

## Sample

The following sample shows how to perform a sign-in third-party services through OAuth 2.0 flows.

Replace `[FLOW]` with the desired sign-in flow, e.g. `users/123/flows/myanimelist-net`.

Replace `[OAUTH2-CODE]` with the **code** provided in the callback URL following a [signin](/credentials/docs/samples/signin) request.

Replace `[OAUTH2-STATE]` with the **state** provided in the callback URL following a [signin](/credentials/docs/samples/signin) request.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
