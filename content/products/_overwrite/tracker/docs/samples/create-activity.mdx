# Create a tracker activity

This page describes how to create a tracker activity.

## Overview

Activities describe at a very granular level an action performed by a user or group of users (i.e. audiences) such as _user X watched episode N from 00:00 to 05:00 on the 20th of September 2022 starting at 2:37pm and finishing at 2:42pm on Netflix_.

## Sample

The following sample shows how to create a tracker activity.

Replace `[TRACKER]` with the resource name of the tracker the activity belong to, e.g. `users/123/trackers/456`.

Replace `[RESOURCE]` with the resource name of the resource (chapters or episodes) to be tracked, e.g. `animes/123/episodes/456`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
