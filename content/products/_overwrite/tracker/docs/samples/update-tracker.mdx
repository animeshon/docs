# Update a tracker

This page describes how to update a tracker.

## Sample

The following sample shows how to update a tracker.

Replace `[TRACKER]` with the resource name of the tracker to be update, e.g. `users/123/trackers/456`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
