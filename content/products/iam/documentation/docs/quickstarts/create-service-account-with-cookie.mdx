# Create service account with cookie

This page describes how to create a service account from the browser cookie.

## Cookie

The cookie can be found by visiting the [Animeshon UI](https://app.animeshon.com/) from a browser and logging into the account. The cookie that will be recognized by the API authentication proxy can be found under the name `ory_kratos_session`.

## Sample

The following sample shows how to create a service account from the browser cookie.

Replace `[NAME]` with the desired service account name, e.g. `users/123/serviceAccounts/my-service-account`.

Replace `[DISPLAY-NAME]` with the desired service account display name, e.g. `My Service Account`.

Replace `[DESCRIPTION]` with the desired service account description, e.g. `A test service account.`.

Replace `[COOKIE]` with the browser cookie, e.g. `ory_kratos_session=ABC123`.
