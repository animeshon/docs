# Create a batch of playlist items

This page describes how to create a batch of playlist items.

## Sample

The following sample shows how to create a batch of playlist items.

Replace `[PLAYLIST]` with the resource name of the playlist that the item belongs to, e.g. `users/123/playlists/456`.

Replace `[RESOURCE-1]`, `[RESOURCE-2]`, and `[RESOURCE-3]` with the resource names of the multimedia resources to be added, e.g. `animes/123`.

:::tip

Tip: do not forget to replace `CLIENT-ID` and `CLIENT-SECRET` with valid [IAM Service Account](/authentication/service-accounts) client credentials.

:::
